X-SYS is a tool used to gain access to any Windows System.

Installation Guide:

1.

Flash the X-SYS.iso on a USB Stick

2.

Boot From the USB Stick (done by pressing F12 at startup or holding shift befor clicking on restart)

3.

Spam the shift key when back in the Login Screen, and start a new task called "install", wait for the pc to restart and you got access

Builder User Manual:

Only use the builder if you know what your doing!

On the Desktop will be the main builder, it will copy anything from the "%appdata%\X-SYS\Scripts" Folder into the mounted image, from the usb stick. You can edit the startnet.cmd to copy your own files from the mounted image to the drive you want to get access to.
